# 安安的毕业设计
```
|src
 |-api 接口管理
 |-components 组件
 |-config 路由配置
 |-layouts 布局
 |-mock 模拟请求
 |-router 路由简单封装
 |-store 状态管理
 |-untils 工具
 |-views 展示

```


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```
